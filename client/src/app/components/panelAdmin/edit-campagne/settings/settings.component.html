Comment souhaitez-vous que le candidat soit chronométré ?

UN CHRONOMÈTRE PAR QUESTION Chaque question sera chronométrée. Une fois le test démarré, les questions se succèderont selon l’ordre défini. Le candidat ne pourra pas revenir à une question précédente. Nous conseillons ce mode pour obtenir un aperçu des connaissances théoriques et pratiques des candidats. (recommandé)
UN CHRONOMÈTRE GLOBAL Un temps global est alloué au candidat pour réaliser l’ensemble du test. Le candidat peut gérer son temps comme bon lui semble : il peut répondre aux questions dans n’importe quel ordre et les modifier plusieurs fois. Nous conseillons ce mode pour évaluer, uniquement, les connaissances pratiques des candidats (lors d’exercices de code par exemple). Temps alloué pour ce test (minutes) x temps
